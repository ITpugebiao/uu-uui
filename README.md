# uu-uui 组建库

 ### 快速开始

 #### 1. 安装组建库

```bash
    npm i uu-uui
```

#### 2. 引入组建库
```javascript
    // 全部引入
    import 'uu-uui/dist/css/index.css';
    import uUI from 'uu-uui';
    Vue.use(uUI);

    // 按需引入
    import 'uu-uui/dist/css/uu-button.css';
    import {uuButton} from 'uu-uui';
    Vue.use(uuButton);
```
